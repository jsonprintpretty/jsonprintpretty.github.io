About

JSONPrettyPrint Online Tool is created for quick use for developers for validating/formatting/minify/pretty print JSON/XML data.
This tool does not store any data(JSON/XML) to server, so that it will help us to validate any JSON/XML and print in a formatted way quickly for use.
No User login/access are required to perform these operations.

Any feedback about the tool can be shared to the admin e-mail id, so that we can update back on the same.
